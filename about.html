About This Blog

Welcome to My Serverless Blog, a demonstration project showcasing the power of AWS serverless architecture. This blog is built using:

  • Amazon S3 for static website hosting
  • AWS Lambda for backend processing
  • Amazon DynamoDB for data storage
  • Amazon API Gateway for RESTful APIs
  • Amazon Cognito for user authentication
  • CI/CD pipelines for automated deployments

This project demonstrates how to build a fully-featured, scalable web application without managing any servers!
